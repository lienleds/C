# Character Variables

In this section, you will learn about character variables in C.

## Topics

- Declaring Character Variables
- ASCII Values
- Format Specifiers

## Example

```c
#include <stdio.h>

int main() {
    char initial = 'A'; // Character variable

    printf("Initial: %c\n", initial);
    printf("ASCII Value: %d\n", initial);

    return 0;
}
```
