# Float Variables

In this section, you will learn about float variables in C.

## Topics

- Declaring Float Variables
- Precision of Floats
- Format Specifiers

## Example

```c
#include <stdio.h>

int main() {
    float pi = 3.14; // Float variable

    printf("Value of pi: %.2f\n", pi);

    return 0;
}
```
